# 🤖 AI Poem Generator using n8n

This project is an automated AI workflow built using n8n that generates poems based on user input and stores them in Airtable.

## 🚀 Features

* Collects user input via form
* Routes data using conditional logic
* Generates AI-based poems using Google Gemini
* Stores output in Airtable database

## 🛠️ Tech Stack

* n8n (workflow automation)
* Google Gemini API
* Airtable

## 🔄 Workflow

1. User submits form
2. Data stored in Airtable
3. Workflow routes based on conditions
4. AI Agent generates poem
5. Output saved back to Airtable

## 📂 Files

* `ai-poem-generator.json` → n8n workflow export

## ⚙️ Setup

1. Import JSON into n8n
2. Add your API keys:

   * Google Gemini API
   * Airtable API
3. Run the workflow

## 📌 Note

This project demonstrates building an AI agent workflow with automation and real-time data handling.
